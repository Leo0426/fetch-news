凭据配置
B站 Bilibili
全局 Cookie 公共路由（视频/动态），无需登录，填 buvid3/buvid4 即可 DevTools → Application → Cookies → api.bilibili.com · 环境变量 BILIBILI_COOKIE 优先级更高
UID 专属 Cookie （关注动态/视频/专栏路由，需要已登录 Cookie）
获取方式： 打开 B站动态 API → DevTools → Network → 点 dynamic_new → 复制 Cookie header。
动态路由需完整 Cookie；视频/专栏路由只需 SESSDATA。 环境变量 BILIBILI_COOKIE_<uid> 优先级更高。
X Twitter / X
Cookie 必须包含 auth_token 和 ct0 DevTools → Application → Cookies → x.com · 环境变量 TWITTER_COOKIE 优先级更高
